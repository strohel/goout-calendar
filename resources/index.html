GoOut Calendar Exporter

This is a simple proxy Web service that serves events of a given user of GoOut.net in an iCalendar format. See homepage on GitHub for more info and source code.

Determine your GoOut textual ID

  1. Navigate to GoOut profile page.
  2. Log-in if you aren't already.
  3. The page redirects you to https://goout.net/en/profile/your-name/ydtc/
  4. Here, ydtc is your textual GoOut ID.
